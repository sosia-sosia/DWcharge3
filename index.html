📜 ルール説明
⚔️ 基本ルール
・プレイヤーHP: 30（最大HP）
・マナはターン開始時に必ず3に固定（蓄積なし）。最大10
・毎ターン5枚ドロー（最初のターンは5枚）
・手札は毎ターン決まった枚数補充され、ターン終了時には全て捨札になる。
　山札がなくなった場合、捨札のカードを全てシャッフルし新たな山札となる。
・シールドは自分のターン開始時に30%減少する（切り捨て）
・カードをタップすると拡大表示され、決定ボタンで使用
・ターン終了ボタンで敵のターンへ
⚡ チャージシステム
ゲーム開始前に1つ選ぶ特殊能力。カウンターが満タンになったらチャージゲージをタップして手動発動。
・格闘家: カード使用3回→このターン攻撃力+4 + ドロー1
・重戦士: コスト2以上のカード使用2回→次の攻撃が2倍 + マナ+2
・僧侶: 回復カード2回→HP+12回復 + ドロー1
・守護者: 盾獲得ごと3回→次の敵からの被ダメを無効 + マナ+1
・錬金術師: 毒カード2回→次の攻撃カードの与ダメ分の毒を付与
・祈祷師: ターン終了2回→マナ+3 + ドロー2 + デッキから1枚選んでコストなしで使用し手札にも加える
・狂戦士: 盾0でターン終了2回→手札全コスト0 + このステージ中ターン開始マナ+1（累積）
⏳ 呪縛タイマー
6ターン目の終了時から呪縛が発動し、ターン終了時にダメージを受ける。
ダメージ = (現在ターン - 5) × 5
例：6ターン目終了→5ダメ、7ターン目終了→10ダメ…
右下の⏳ボックスで現在ターンと状況を確認。長押しで詳細。
⚔️ ダメージ計算
・攻撃力の加算（+）は乗算（×）より先に計算される。
　例：基本攻撃5、攻撃+5、攻撃×2の場合
　　　(5 + 5) × 2 = 20ダメージ
・「次の攻撃が2倍」の効果は、双撃・連撃など
　1枚で複数回攻撃するカードでも1撃目にのみ適用される。
　2撃目以降は通常ダメージになる。
🔴 レアリティ
◆ COMMON → ◆ UNCOMMON → ◆ RARE → ◈ EPIC → ★ LEGEND → ⚡ GAUGE
ステージが進むほど高レアリティが出やすくなる。GAUGEはチャージ選択時の初期デッキ専用。
